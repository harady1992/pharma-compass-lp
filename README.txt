ファーマコンパス LP 共有用ファイル

■ 開き方
このフォルダ内の index.html をブラウザで開くとLPが表示されます。

■ 公式LINEリンク
LP内の「招待を受け取る」ボタンは、以下の公式LINE URLに設定済みです。
https://lin.ee/QAYDAs8

■ 公開URLとして使う場合
このフォルダごと、Webサーバー / GitHub Pages / Netlify / Xserver などにアップロードしてください。
index.html と assets フォルダの位置関係は変えないでください。
